Twitter MCP Agent

This project is a personal analytics tool designed to connect to the Twitter API and analyze engagement trends on my own posts. It retrieves tweet metrics (likes, retweets, replies, impressions, etc.) to help optimize future content based on performance insights.

Use Case

  • Monitor account engagement over time
  • Analyze tweet trends and identify high-performing content
  • Track follower growth and active hours
  • All data is limited to my own account and used privately
